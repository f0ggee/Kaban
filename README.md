# Filesbes - Fast File Transfer

Filesbes is a dedicated service designed for the rapid transfer of files between different devices. It provides a streamlined alternative to using messaging applications or email services for simple document sharing.

We highly value your privacy. The platform is built on the principle of ephemeral data handling, ensuring your information is never stored longer than necessary.


---

## How It Works

The transfer process is designed to be completed in four steps:
1. Upload: Select and upload your file to the platform.
![exampleDownloading](https://github.com/user-attachments/assets/cb346c1f-cc3c-4e01-8264-07f5c4c44adb)
2. Link: Receive a unique, personal transfer link.
![GettingLink](https://github.com/user-attachments/assets/4a1b7488-241a-4f9b-a11e-13d1c366bff0)
3. Access: Open the provided link in a web browser on the receiving device.
4. Download: Save the file to the new device.

---

## Operational Logic and Security

Filesbes operates with strict data retention policies to ensure security and privacy:

* Transit-Only System: This service is not a cloud storage provider. It functions exclusively as a bridge for data transfer.
* Immediate Erasure: Files are automatically and permanently deleted from the system immediately after the first successful download.
* Encryption Timeout: If the encryption option is selected, the file will be deleted after 5 minutes if it has not been accessed.
* Global Auto-Deletion: In all cases, any remaining files are erased from the server 24 hours after the initial upload.

## Technical Features

The project is built with a focus on performance and cross-platform compatibility:

* Responsive Design: Uses CSS clamp logic to ensure the interface is optimized for all screen sizes, from mobile phones to desktop monitors.
* Typography: Integrated with the Inter font family for high legibility and a modern aesthetic.
* Localization: Native support for English and Russian languages based on user browser preferences or manual selection.
* Minimalist UI: A dark-themed terminal style interface designed for speed and clarity.
